# Habit Hubby

A full-stack web application with a Django backend and React frontend.

## Overview

This project is a full-stack web application using Django for the backend API and React for the frontend user interface.

## Features

- RESTful API built with Django REST Framework
- Frontend built with React and modern JavaScript
- User authentication and authorization
- Responsive UI design

## Technologies Used

- Backend: Django, Django REST Framework, Python
- Frontend: React, JavaScript, HTML, CSS
- Database: SQLite (default) / can be configured
